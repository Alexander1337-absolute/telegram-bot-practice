Описание проекта

Проект представляет собой создание Telegram-бота, который помогает автоматизировать задачи и улучшить взаимодействие с пользователями. В рамках данной работы мной самостоятельно был разработан функционал бота, написан код и проведено тестирование. Этот проект позволил применить знания по программированию, работе с API и управлению проектами.
